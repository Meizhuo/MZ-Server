## 错误码列表

必要的时刻才开启一个特定的错误码，否则使用通过错误码40000,然后错误信息自定义
40000为通用错误

**Note: 0001<=xxx<=990**

error_code范围 | 含义
-----------	   | ------
40xxx|请求错误:例如请求方式错误
41xxx|不合法xxx(例如邮箱格式错误)
42xxx|缺少XXX参数
43xxx|应用逻辑错误(如账号冻结)


error_code	 | description  
------------ | ------------- 
40000 | 通用的
40001 | 请求方法错误
~|~
43001 | 账号被冻结
