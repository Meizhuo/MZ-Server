用户Admin
===
`/home/admin/*`



## 注册    
`POST /home/institution/register` 

字段	|描述 |  是否必须 
------------ | -------------| -------------
nickname | 用户昵称  	| Y
phone| 手机号码 | N
email|  邮箱  	| Y
psw  | 密码 8-16位数字or字母 	| Y


**Response**  

```json
{
	"code":20000,
	"response"："register success"
}
```
